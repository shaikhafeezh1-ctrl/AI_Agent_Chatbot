Features
*  Search Agent – Retrieves relevant information from external sources.
*   Reader/Scraper Agent – Extracts and processes structured/unstructured data.
*   Writer Chain – Generates coherent, context-aware responses.
*   Critic Chain – Evaluates and refines outputs for accuracy and clarity.
*   Streamlit UI – Interactive interface for seamless user experience.
*   Deployment – Hosted on Hugging Face Spaces for easy access and sharing.

Tech Stack
* Python
* LangChain (agents, chains, tools)
* Streamlit (UI)
* Hugging Face Spaces (deployment)
* OpenAI / Hugging Face LLMs (backend models)

Use Cases

* Automated Q&A
